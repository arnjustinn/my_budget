 # MyBudget

A simple budget tracking app built to help users manage their income and expenses.  
It allows adding, editing, and deleting transactions, and provides a clear summary of total balance, income, and spending.

## Features
- Add and manage income and expenses
- View balance summary
- Simple and clean interface

## Technologies Used
- Flutter
- Dart

## How to Run
1. Open the project in your code editor or terminal  
2. Run the following command to install dependencies:
'flutter pub get'
3. Then start the app:
'flutter run'
